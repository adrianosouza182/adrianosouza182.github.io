Detailed guide: Masternodes

To start IMG masternode on a dedicated server you will need:

  • 10000 IMG collateral
  • server with Ubuntu (18.04)
  • IMG Core on local machine
  • basic Linux skills

Setup VPS

Prepare the system:

sudo apt-get update
sudo apt-get upgrade 
sudo apt-get install ufw
sudo apt-get install build-essential libtool autotools-dev automake pkg-config libssl-dev libevent-dev bsdmainutils
sudo apt-get install libboost-all-dev
sudo add-apt-repository ppa:bitcoin/bitcoin
sudo apt-get update
sudo apt-get install libdb4.8-dev libdb4.8++-dev
sudo apt-get install libminiupnpc-dev libzmq3-dev

Install IMG on VPS

git clone https://github.com/mceme/ImageCoin.git
cd ImageCoin
cd imginstall
bash img-install.sh

 

Go to your local datadir and edit masternode.conf to make it look like example below

 

alias IP:port masternodeprivkey collateral_output_txid collateral_output_index

check status

ImageCoin-cli getinfo
ImageCoin-cli mnsync status
ImageCoin-cli masternode status

systemctl status ImageCoin #To check the service is running.  
systemctl start ImageCoin #To start ImageCoin service.  
systemctl stop ImageCoin #To stop ImageCoin service.  
systemctl is-enabled ImageCoin #To check whetether ImageCoin service is enabled on boot or not.
